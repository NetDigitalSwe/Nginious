Chapter 3. Installing

3.1. Prerequisites

Nginious requires a Java runtime environment version 1.6 or higher to be installed for running the server. If you plan to develop web applications with Nginious you need a Java JDK version 1.6 or later. To use the Eclipse plugin you need Eclipse Indigo or later.

3.2. Download

You can download Nginious from the Nginious web site.

3.3. Unpacking

The Nginious distribution is packed as a gzipped tar file or as a zip file depending on which distribution you have downloaded. All distributions contain the following contents.

  • The Nginious jar libraries and start scripts

  • The users guide

  • The API documentation

3.4. Eclipse plugin

You can install the Nginious plugin using the Eclipse update manager. The plugin can be downloaded from Nginious plugin website. The plugin contains all the necessary components to develop web applications for Nginious in Eclipse.
