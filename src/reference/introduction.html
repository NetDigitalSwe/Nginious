Chapter 1. Introduction

Nginious is an application server which removes the need for redeployment. It has the following main features.

  • Ability to detect code and resource changes runtime and reload them.

  • Full support for all versions of the HTTP protocol built on top of a asynchronous I/O engine.

  • Support for web applications similar to the ones found in servlet based application servers.

  • Web socket support.

  • Support for RESTful services with automatic conversion of beans to and from JSON and XML.

  • XSP - template language with an ultra fast compiler which compiles pages directly into Java classes.

  • Built in support for handling multipart content efficiently.

  • Integration with Eclipse through a plugin.

  • Deployment and rollback of web applications to a production site from command line or Eclipse.

  • Statistics!

Chapter 2, Overview provides a detailed overview of Nginious.
