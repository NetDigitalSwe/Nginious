Chapter 13. Ant integration

13.1. Compiling XSP templates

The xspc ant task is used for precompiling XSP templates into Java classes for inclusion into web applications. It works in a similar way as the javac task for compiling java classes. Below is an example that compiles a set of XSP templates into Java classes. Note that the jar file nginious-ant-1.0.0.jar that includes has to be included in the classpath. 


<taskdef name="xspc" classname="com.nginious.http.ant.XspcTask" />
						
<xspc srcdir="src/main/resources/xsp" destdir="build/webapp/WEB-INF/classes" />
Attribute Description
srcdir Location where XSP templates are found
destdir Location where compiles HTTP service classes should be placed.

13.2. Deploying

The deploy task deploys a web application archive to a remote Nginious server. The jar file nginious-ant-1.0.0.jar that contains ant tasks has to be included in the classpath. 


<taskdef name="deploy" classname="com.nginious.http.ant.DeployTask" />
						
<deploy url="http://www.nginious.com/admin/application/test"
    file="build/libs/test.war"
    username="admin"
    password="admin" />
Attribute Description
url URL of Nginious REST admin controller for deploying web applications. The last path element of the URL matches the name of the web application being deployed.
file The web application archive to deploy.
username Username for accessing remote Nginious server admin controller.
password Password for accessing remote Nginious server admin controller.
